Política de Privacidade

Última atualização: junho de 2025

⚡ Resumo simples: o PhotoClean não armazena suas fotos, não vende seus dados e não guarda nada em servidores próprios. Tudo fica na sua conta Google.

1. O que é o PhotoClean

O PhotoClean é uma ferramenta gratuita que usa inteligência artificial para ajudar você a identificar e remover fotos desnecessárias do seu Google Photos — como screenshots, fotos borradas, duplicatas e memes antigos.

2. Quais dados acessamos

Para funcionar, o PhotoClean solicita acesso à sua conta Google com as seguintes permissões:

  • Google Photos (leitura): para listar e visualizar suas fotos
  • Google Photos (exclusão): para mover fotos para a lixeira quando você solicita
  • Perfil básico: nome e foto de perfil, apenas para exibir na interface

3. O que fazemos com seus dados

  • As fotos são acessadas apenas para análise temporária — baixamos um thumbnail pequeno (512×512), enviamos para a API do Gemini e descartamos imediatamente
  • Não armazenamos suas fotos em nenhum servidor
  • Não vendemos seus dados para terceiros
  • Não compartilhamos suas informações com ninguém, exceto o Google (para autenticação) e a API Gemini (para análise)

4. API do Gemini (Google)

A análise das fotos é feita pela API Gemini da Google. Os thumbnails enviados para análise estão sujeitos à Política de Uso da API Gemini. A chave de API é fornecida pelo próprio usuário e processada apenas durante a sessão ativa.

5. Armazenamento local

O PhotoClean salva no seu próprio navegador (localStorage) apenas:

  • O resultado das análises já realizadas (para não repetir)
  • Preferências de sessão

Esses dados ficam somente no seu dispositivo e podem ser apagados limpando os dados do navegador.

6. Segurança

A sua chave da API Gemini é criptografada (AES-256) e armazenada apenas em um cookie de sessão seguro. Ela nunca é registrada em logs nem enviada para terceiros que não sejam a própria API do Google.

7. Exclusão de dados

Como não armazenamos dados em servidores, não há nada para solicitar exclusão. Para remover os dados locais, basta limpar o histórico e dados do site no seu navegador.

Para revogar o acesso do PhotoClean à sua conta Google, acesse: myaccount.google.com/permissions

8. Crianças

O PhotoClean não é direcionado a menores de 13 anos e não coleta dados de crianças intencionalmente.

9. Alterações nesta política

Podemos atualizar esta política eventualmente. A data no topo desta página indica a versão mais recente. O uso continuado do serviço após alterações constitui aceitação da nova política.
